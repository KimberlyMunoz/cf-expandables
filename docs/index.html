  • Structural cheat sheet:
-----------------------
.expandable
  .expandable_target
    .expandable_cue-open
    .expandable_cue-close
  .expandable_content
  • This is a barebones expandable pattern which our official recommended expandable is based off of. Please only use this is if you need a custom expandable experience.
  • Use the anchor tag for the .expandable_target element to allow for keyboard access. The href attribute should be set to the ID of the .expanded_content element as a fallback for no JavaScript support. When pressed the link will jump you down to the content. Admittedly this is a very minor convenience in most cases, but in some cases there might actually be a significant amount of content between .expandable_target and .expandable_content.
  • The aria-pressed attribute on .expandable_target gets automatically added and updated by JavaScript.
